# SensorConnect: Editable Parameters for Node 16904

Source: screenshots from SensorConnect (TC-Link-200-OEM, Node 16904).

## 1) Transducer Type
- Thermocouple
- RTD
- Thermistor

## 2) Sensor Type (for RTD)
- Uncompensated (Resistance)
- PT10/
- PT50
- PT100
- PT200
- PT500
- Other RTD types are present further down in the dropdown list.

## 3) Wire Type
- 2 Wire
- 3 Wire
- 4 Wire

## 4) Low Pass Filter / Filter Cutoff
- 294 Hz
- 12.66 Hz (92db 50/60 Hz rejection)
- 2.6 Hz (120db 50/60 Hz rejection)

## 5) Input Range
- +/-1.35 V or 0 to 1 mega-ohms (Gain: 1)
- +/-1.25 V or 0 to 10000 ohms (Gain: 2)
- +/-625 mV or 0 to 3333.3 ohms (Gain: 4)
- +/-312.5 mV or 0 to 1428.6 ohms (Gain: 8)
- +/-156.25 mV or 0 to 666.67 ohms (Gain: 16)
- Additional ranges are available lower in the dropdown list.

## 6) Linear Calibration Units8529 
- Temperature (ch1): Ohm, Milliohm, Kiloohm
- CJC Temperature (ch2): Celsius, Fahrenheit, Kelvin

## 7) Sampling Tab
- Lost Beacon Timeout: On/Off + value in minute(s)
- Diagnostic Info Interval: On/Off + value in second(s)
- Storage Limit Mode: Stop, Overwrite

## 8) Power Tab
- Default Operation Mode: Idle, Sleep, Sample
- User Inactivity Timeout: On/Off + value in second(s)
- Check Radio Interval: value in second(s)
- Transmit Power: 10 dBm, 5 dBm, 0 dBm
